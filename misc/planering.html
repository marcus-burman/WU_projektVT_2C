3. Dokumentation Jag har registrerat mig på github och laddat ner github desktop samt forkat uppgiften ifrån Leos github och sparat ner den på mitt konto. Jag har sen skapat alla filer som behövs för uppgiften som t.ex. script.js och style.css. När detta var klart började jag med min hamburgar meny för att få den att fungera då det är något jag vill ha på min hemsida. Mestadels av det jag gjorde idag gick bra då jag än så länge inte stött på några problem gällande kodningen. Något att ta med sig till nästa lektion är att ta bättre tillvara på tiden då jag hade fått mer gjort om jag hade arbetat mer effektivt under lektionen. Jag ska även ta med mig hur man placerar saker oberoende av annat med position fixed samt att använda procent istället för pixlar då det blir lättare att optimera hemsidan för båda telefon och dator.
